## Summary

**Total new synonyms suggested: ~200+**

**Priority order:**

1. **4 biomarkers with NO synonyms** (Bicarbonate, Monocytes, Potassium, Sodium)
2. **25 biomarkers with only 2 synonyms**
3. **Remaining biomarkers** to add more comprehensive coverage

## Notes

- Many variations include common abbreviations, alternative spellings, and French translations
- Some synonyms include unit abbreviations (e.g., "MCH pg") which may appear on lab reports
- Consider case-insensitive matching or normalization when implementing
- Some biomarkers might have method-specific names (e.g., "eGFR CKD-EPI") - you may want to handle these separately
